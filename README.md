# Computer-Prototype-378

## Build and Run Instructions ##
`cd Computer-Prototype-378/client/`
`npm install`
`npm run dev`

## Tasks to be Completed ##

### Task 1 ###
User will be able to set a weight goal and view their progress at any given point.

### Task 2 ###
User will be able to view the caloric value of meal items.

### Task 3 ###
User will be able to set preferences so that allergies and dislikes are handled properly.
